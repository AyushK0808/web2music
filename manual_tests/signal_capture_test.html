Scroll & Move Around

This page measures cursor and scroll speed using the same formulas your spec defines for content_script.js: distance divided by time, averaged over the last 5 readings, with throttling so it doesn't run hundreds of times per second.

Move your mouse slowly, then quickly — watch "Cursor speed" react. Then scroll slowly, then rapidly — watch "Scroll speed" react separately. The colour changes from green to yellow to red as speed increases, roughly matching the thresholds your B2 code already uses (scroll > 800 px/s, cursor > 600 px/s).

The two "events/sec" counters show how many times the throttle actually let an event through in the last second — this proves the throttling itself is working, capping mousemove at 20/sec and scroll at 10/sec no matter how fast the raw browser events fire.
